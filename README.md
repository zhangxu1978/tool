# 数字竞技场工具

一个基于Web的工具，用于管理音频、图片文件和对话系统。

## 功能特点

- 音频文件管理（BGM和音效）
- 图片文件管理
- 对话管理和预览
- 图片编辑工具
- 宗门管理
- 灵脉管理
- 道具管理
- 角色管理
- 数字竞技场

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动应用

```bash
npm start
```

然后在浏览器中访问 http://localhost:3000

### 开发模式

```bash
npm run dev
```

使用nodemon自动重启服务器，方便开发调试。

## 使用说明

1. 启动服务器后，在浏览器中打开 http://localhost:3000
2. 使用顶部导航栏切换不同的功能模块
3. 上传和管理音频、图片文件
4. 编辑和预览对话内容
5. 管理游戏相关数据

## 技术栈

- 前端：HTML5, CSS3, JavaScript, Bootstrap 5
- 后端：Node.js, Express
- 文件上传：Multer

## 目录结构

```
├── public/          # 前端文件
│   ├── css/        # 样式文件
│   ├── js/         # JavaScript文件
│   └── *.html      # HTML页面
├── data/           # 数据文件
├── img/            # 图片文件
├── sound/          # 音频文件
│   ├── bgm/        # 背景音乐
│   └── sfx/        # 音效
└── server.js       # 服务器主文件
```

## 故障排除

如果遇到问题：
1. 确保Node.js版本 >= 14
2. 检查端口3000是否被占用
3. 确保所有依赖都已正确安装
