数字竞技场工具 - 启动说明

1. 安装依赖：
   npm install

2. 启动服务器：
   npm start

3. 在浏览器中访问：
   http://localhost:3000

4. 开发模式（自动重启）：
   npm run dev

5. 单独启动卡片管理服务器：
   node card-server.js
   或双击 start-card-server.bat

功能模块：
- 音频管理：上传、管理BGM和音效文件
- 图片管理：上传、管理角色和背景图片
- 对话管理：创建和编辑游戏对话
- 图片编辑：图片切片和组合工具
- 宗门管理：管理游戏中的宗门数据
- 灵脉管理：管理灵脉模板数据
- 道具管理：管理游戏道具数据
- 角色管理：管理游戏角色数据
- 卡片管理：管理游戏卡片数据（新增）
- 数据管理：统一的数据管理界面

注意：
- 确保Node.js版本 >= 14
- 确保端口3000没有被占用
- 所有数据文件存储在data目录中
- 卡片管理可以独立运行在端口3003
